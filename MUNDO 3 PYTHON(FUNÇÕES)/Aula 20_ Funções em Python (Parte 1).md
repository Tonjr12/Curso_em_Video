# ⚙️ Aula 20: Funções em Python (Parte 1)

Este repositório contém a prática dos conceitos fundamentais de funções (rotinas) em Python, baseados no curso do Curso em Vídeo.

## 🚀 Conceitos Abordados
- **Criação de Rotinas**: Uso de `def` para encapsular blocos de código repetitivos (Ex: Cabeçalhos, somas).
- **Parâmetros**: Passagem de dados para funções.
- **Empacotamento (`*args`)**: Técnica para receber um número indefinido de parâmetros utilizando `*num`.
- **Manipulação de Mutáveis**: Como funções podem alterar diretamente o conteúdo de listas passadas como argumento.

## 🛠️ Scripts Desenvolvidos
- `aula20_FUNÇÕES(exemplo).py`: Criação de cabeçalhos dinâmicos.
- `aula20_FUNÇÕES(exemplo-a).py`: Função de soma simples.
- `aula20_FUNÇÕES(exemplo-b).py` e `-c.py`: Uso de desempacotamento para múltiplos valores.
- `aula20_FUNÇÕES(exemplo-d).py`: Função que dobra valores em uma lista (mutabilidade).
- `aula20_FUNÇÕES(exemplo-e).py`: Função de soma com múltiplos parâmetros desempacotados.

## 🧠 Aprendizado Chave
Funções são o primeiro passo para a **modularização** do código. Em vez de escrever todo o fluxo principal, dividimos o problema em pequenas partes reutilizáveis.
